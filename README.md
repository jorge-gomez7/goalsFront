# 🎯 Goals Tracker App

Una aplicación web desarrollada con **React** que permite agregar y visualizar metas (goals) con su nombre, descripción y fecha límite. Diseñada para ser completamente responsiva, con una vista optimizada para escritorio y móvil.

---

## 🚀 Funcionalidades

- Agregar tareas con nombre, descripción y fecha.
- Visualización en formato de tarjetas (cards).
- Scroll lateral para mantener el formulario visible en escritorio.
- Menú tipo hamburguesa en versión móvil para mostrar el formulario.
- Diseño limpio y adaptable.
